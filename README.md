# App de Apps
Esta aplicación fue adaptada por el grupo Semillero de Innovación Geográfica (SIG) de Esri Colombia. Para mayor información consulte www.esri.co o visite @geo_geeks en twitter.
Este proyecto contiene ejemplos para el ArcGIS Runtime SDK para .NET (Windows Phone 8.1)

## Instrucciones 

Verifique que la configuración del sistema sea la correcta para el API de Windows Phone en el ArcGIS Runtime SDK for .NET:
  * [Windows Phone](http://developers.arcgis.com/net/store/guide/system-requirements.htm).
  * [ArcGIS Runtime SDK for .NET](http://esriurl.com/dotnetsdk) 

Esta aplicación hace referencia al uso de un paquete Nuget. Este se descarga automáticamente cuando la aplicación es construida por primera vez.
Una vez abra la aplicación asegurese de seleccionar la plataforma adecuada. Si va a probar desde un dispositivo seleccione una arquitectura ARM. Si va a probar desde un emulador use una arquitectura x86.
In Visual Studio, open the solution for the Desktop, Phone, or Store samples.
